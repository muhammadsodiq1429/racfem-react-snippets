# Racfem React Snippets

Bu extension React'da tezkor funksional komponentlar yaratish uchun mo'ljallangan snippetlarni o'z ichiga oladi.

## Foydalanish

`.jsx`, `.js`, `.tsx`, yoki `.ts` faylida **racfem** deb yozib, `Enter` tugmasini bosing.
